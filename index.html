AssetLibraryTools

AssetLibraryTools is a blender add-on which I developed to assist in the creation of blender asset libraries. This add-on was the first of its kind and has received well over 1000 downloads on gumroad. (It was so early that I wrote it before some of the API functions were even documented!)

Vid2Photogrammetry

Vid2Photogrammetry is software designed to extract a high-quality dataset of images for the creation of 3D models via photogrammetry from an input video. This was my A-Level computer science project.
Written in C++, utilises OpenCV.
